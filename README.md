# CUET SPHERE

## Clone Repository
```bash
git clone https://github.com/CUET-SPHERE/CUET-SPHERE.git
```

## Setup Frontend
```bash
# Navigate to frontend directory
cd Frontend

# Install dependencies
npm install

# Run development server
npm run dev
```
